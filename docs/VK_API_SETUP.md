# 🔑 ПОЛУЧЕНИЕ VK API ТОКЕНА - ПОДРОБНАЯ ИНСТРУКЦИЯ

Эта инструкция поможет вам получить токен доступа к VK API для работы приложения VK Publisher.

---

## 📋 ЧТО ВАМ ПОНАДОБИТСЯ

1. Аккаунт ВКонтакте
2. Созданное сообщество (группа или публичная страница), где вы являетесь администратором

---

## 🚀 ШАГ 1: СОЗДАНИЕ ПРИЛОЖЕНИЯ VK

### 1.1 Перейдите на портал разработчиков VK
Откройте в браузере: **https://dev.vk.com/**

### 1.2 Войдите через свой аккаунт VK
Нажмите кнопку **"Войти"** в правом верхнем углу и авторизуйтесь своим аккаунтом ВКонтакте.

### 1.3 Перейдите в раздел "Мои приложения"
В верхнем меню нажмите на **"Приложения"** → **"Мои приложения"**

Или перейдите напрямую: **https://dev.vk.com/my-apps**

### 1.4 Создайте новое приложение
Нажмите кнопку **"Создать приложение"** (обычно синяя кнопка справа).

### 1.5 Выберите тип приложения
- В поле **"Название"** введите любое имя, например: `VK Publisher Bot`
- В поле **"Платформа"** выберите **"Standalone-приложение"** (это важно!)
- Нажмите **"Создать"**

> ⚠️ **ВАЖНО:** Обязательно выбирайте **Standalone-приложение**, иначе токен не будет работать!

### 1.6 Подтвердите создание
Вам может потребоваться подтвердить действие через SMS или код из приложения VK.

## 🔐 ШАГ 2: ПОЛУЧЕНИЕ ТОКЕНА ДОСТУПА

### Способ А: Через настройки приложения (РЕКОМЕНДУЕТСЯ)

#### 2.1.1 Откройте настройки приложения
После создания приложения вы попадете на страницу с настройками. Если нет - найдите свое приложение в списке и кликните по нему.

#### 2.1.2 Найдите раздел "Настройки OAuth"
Прокрутите страницу вниз до раздела **"Настройки OAuth"**.

#### 2.1.3 Скопируйте ID приложения
В поле **"ID приложения"** вы увидите цифры - это ваш `APP_ID`. Запишите его.

#### 2.1.4 Получите токен через браузер
Откройте новую вкладку и вставьте следующий URL, заменив `YOUR_APP_ID` на ваш ID приложения:

```
https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,photos,offline&response_type=token&v=5.199
```

**Пример для APP_ID = 12345678:**
```
https://oauth.vk.com/authorize?client_id=12345678&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,photos,offline&response_type=token&v=5.199
```

#### 2.1.5 Разрешите доступ
- Нажмите **"Разрешить"** когда появится запрос на доступ
- Вас перенаправит на страницу с пустым содержимым (blank.html)
- Посмотрите на адресную строку браузера - она будет выглядеть так:

```
https://oauth.vk.com/blank.html#access_token=abc123def456...&expires_in=0&user_id=123456789
```

#### 2.1.6 Скопируйте токен
Скопируйте значение после `access_token=` до знака `&`. Это и есть ваш токен!

**Пример:**
```
access_token=abc123def456ghi789jkl012mno345pqr678stu901vwx234yz
```
Ваш токен: `abc123def456ghi789jkl012mno345pqr678stu901vwx234yz`

---

### Способ Б: Через форму отладки (альтернативный)

#### 2.2.1 Перейдите на страницу отладки
Откройте: **https://dev.vk.com/tools/access-token-checker**

#### 2.2.2 Выберите ваше приложение
Если у вас несколько приложений, выберите нужное.

#### 2.2.3 Выберите права доступа
Отметьте галочками:
- ✅ **Доступ к стене** (wall)
- ✅ **Доступ к фотографиям** (photos)
- ✅ **Расширенный доступ к API** (offline)

#### 2.2.4 Сгенерируйте токен
Нажмите кнопку **"Получить токен"** и скопируйте полученное значение.

## 🏷️ ШАГ 3: ПОЛУЧЕНИЕ ID ГРУППЫ (COMMUNITY ID)

### 3.1 Узнайте ID вашей группы

#### Вариант А: Через страницу группы
1. Откройте вашу группу ВКонтакте
2. Посмотрите на адресную строку:
   - Если адрес вида `https://vk.com/public123456789` - ID это `123456789`
   - Если адрес вида `https://vk.com/my_group_name` - нужно узнать цифровой ID

#### Вариант Б: Через онлайн-сервисы
1. Перейдите на сайт: **https://regvk.com/tools/groupId/**
2. Вставьте ссылку на вашу группу
3. Скопируйте полученный цифровой ID

#### Вариант В: Через API
1. Откройте: **https://dev.vk.com/method/groups.search**
2. В поле `q` введите название вашей группы
3. Нажмите "Выполнить"
4. В ответе найдите поле `id` - это и есть ваш ID группы

> ⚠️ **ВАЖНО:** ID группы должен быть положительным числом (без минуса)!

## 🤖 ШАГ 4: СОЗДАНИЕ TELEGRAM БОТА ДЛЯ МОДЕРАЦИИ

### 4.1 Откройте @BotFather
В Telegram найдите бота **@BotFather** (официальный бот для создания ботов)

### 4.2 Создайте нового бота
1. Напишите `/newbot`
2. Введите имя бота (например: `VK Publisher Moderator`)
3. Введите username бота (должен заканчиваться на `bot`, например: `vk_publisher_mod_bot`)

### 4.3 Скопируйте токен
BotFather выдаст вам токен вида:
```
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz123456
```
Сохраните этот токен!

### 4.4 Узнайте свой Telegram ID
1. Найдите бота **@userinfobot** или **@getmyid_bot**
2. Напишите ему любое сообщение
3. Он пришлет ваш ID (число, например: `987654321`)

## 📝 ШАГ 5: ЗАПОЛНЕНИЕ ФАЙЛА .ENV

После получения всех данных откройте файл `.env` и заполните:

```bash
# VK API
VK_ACCESS_TOKEN=abc123def456...               # Токен из шага 2
VK_GROUP_ID=123456789                         # ID группы из шага 3

# Telegram
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjkl... # Токен бота из шага 4.3
TELEGRAM_MODERATOR_CHAT_ID=987654321          # Ваш ID из шага 4.4

# Остальные настройки можно оставить по умолчанию
```

## ❌ ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

| Проблема | Решение |
|----------|---------|
| Токен не работает | Убедитесь что выбрали Standalone-приложение |
| Токен истекает | Добавьте параметр `offline` в scope при получении |
| Нет доступа к стене | Проверьте что выдали права `wall` при получении токена |
| ID группы отрицательный | Уберите знак минуса, должен быть только номер |
| Бот не отвечает | Проверьте что добавили бота в администраторы канала (если нужно) |
