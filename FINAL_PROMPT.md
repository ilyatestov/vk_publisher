## 🏗️ 1. Архитектура (обязательно)

### Clean Architecture + Dependency Injection
```
/src
  /domain         # Сущности: Post, Account, Media, Analytics
  /application    # UseCases: PostService, SchedulerService, AIService
  /infrastructure # Реализации: VKClient, TelegramClient, OllamaClient, RedisCache
  /interfaces     # API (FastAPI), CLI, Web UI, Telegram Bot
```

### Интерфейс для мультиплатформенности
```python
from typing import Protocol, AsyncResult

class SocialPublisher(Protocol):
    async def publish(self, post: Post) -> PublishResult: ...
    async def validate_credentials(self) -> bool: ...
    async def get_rate_limits(self) -> RateLimitInfo: ...

# Реализации: VkPublisher, TelegramPublisher, DzenPublisher
```

## ⚙️ 2. Очереди, производительность, надёжность

### Асинхронный pipeline
```
fetch → rewrite (Ollama) → moderate → enrich (hashtags, title) → publish → analytics
```

### Требования:
- [ ] **TaskIQ / Celery + Redis** для фоновых задач (отвязать API от долгих операций)
- [ ] **Retry с экспоненциальной задержкой + jitter** для VK API
- [ ] **Circuit Breaker** для внешних сервисов (Ollama, VK API)
- [ ] **Dead Letter Queue** для неудачных публикаций + ручной retry через UI
- [ ] **Идемпотентность**: хэш контента в БД → защита от дублей при сбоях
- [ ] **Redis-кэш** с TTL для частых запросов (`wall.get`, `users.get`)
- [ ] **Rate-limiting middleware** (slowapi) для защиты от блокировок

### Оптимизация под RTX 4060 (8 ГБ VRAM):
- Поддержка GGUF-моделей (Llama 3 8B Q4_K_M, Mistral 7B)
- Асинхронные вызовы к Ollama с таймаутами
- Приоритизация задач: лёгкие → тяжёлые

## 🚀 4. Функционал (приоритеты)

### 4.1 Планировщик + мульти-аккаунты
- `publish_at`, отложенные публикации, cron-расписание
- Поддержка нескольких VK-токенов / групп / массовый постинг

### 4.2 Кросс-постинг
- **Telegram** (обязательно, приоритет №1)
- Яндекс.Дзен, Одноклассники (опционально, через адаптеры)

### 4.3 Умные AI-функции
```python
# Пример: авто-хэштеги
async def generate_hashtags(content: str, category: str) -> list[str]:
    prompt = f"Подбери 5-7 релевантных хэштегов для '{category}':\n{content}"
    return await ollama.generate(prompt, model="llama3:8b")
```
- Авто-заголовки, анализ тональности, A/B-тестирование текстов
- Рекомендации по времени публикации на основе аналитики

### 4.4 Поддержка аудио-контента ⭐ (для музыкальных проектов)
- Интеграция с локальными нейросетями: **YuE**, **ACE-Step** → генерация музыки
- Авто-создание обложек + публикация плейлистов в ВК / Яндекс.Музыка
- Метаданные: исполнитель, жанр, теги → авто-заполнение

### 4.5 Агрегация данных + авто-рерайт
- Парсеры: курсы криптовалют (TON, Solana), банковские кэшбэки, новостные RSS
- Ollama переписывает сухие данные в вовлекающие посты

### 4.6 Аналитика и отчётность
- Метрики: просмотры, лайки, репосты, вовлечённость по постам
- Сравнение источников контента, AI-инсайты ("Посты с вопросами получают на 34% больше комментариев")
- Экспорт отчётов: PDF/CSV
